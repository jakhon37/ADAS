                                                    Prerequisites

```bash
# Install test dependencies
pip install pytest
```

                                                      Run Tests

```bash
# 1. Run all tests
pytest tests/ -v

# 2. Run all tests (quiet mode)
pytest tests/ -q

# 3. Run specific test file
pytest tests/test_safety.py -v

# 4. Run specific test function
pytest tests/test_safety.py::test_safety_monitor_speed_limit -v

# 5. Run with coverage
pip install pytest-cov
pytest --cov=src/adas tests/

# 6. Run and stop at first failure
pytest tests/ -x

# 7. Run with detailed output
pytest tests/ -vv

# 8. Run using make (if you have it)
make test
```
